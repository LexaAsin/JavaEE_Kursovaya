📚 Информационная подсистема учёта выдачи книг

Веб-приложение на чистой Java EE для автоматизации работы библиотеки: учёт книг, авторов, пользователей и выдачи книг.

🎯 О курсовой работе

Учебный проект — курсовая работа студента группы ПИ‑331 Асина А.А.

Приложение реализует *полный цикл* библиотечного обслуживания:

- регистрация и аутентификация (роли **ADMIN** / **USER**)
- управление книгами и авторами
- управление пользователями системы
- выдача книг читателям с выбором срока (1, 2 или 4 недели)
- досрочный возврат книг
- разграничение прав доступа (администратор видит всё, пользователь — только свои книги)
